# SymfonyPress

## Installation

```
git clone https://github.com/wiejakp/SymfonyPress.git
cd SymfonyPress/
composer install
```

## Reset

```
sudo rm -r public/ private/ composer.lock vendor/
git reset --hard origin master
```

Navigate to: [http://symfonypress.dev/symfony](http://symfonypress.dev/symfony)
